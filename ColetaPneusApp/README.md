# ColetaPneusApp

App para conexão entre clientes que precisam descartar pneus e prestadores que realizam a coleta

## 🎯 FLUXOS PRINCIPAIS
- Cadastro de usuário | Solicitação de coleta | Aceite por prestador | Confirmação de coleta

## 🛠️ TECNOLOGIAS ESCOLHIDAS
- Kotlin (linguagem principal - fundamentos sólidos)
- Android Jetpack (componentes padrão do Android)
- Firebase Realtime Database (simples e adequado para volume inicial)

## 📂 ESTRUTURA DO PROJETO
- app/src/main/java: Código Kotlin organizado por pacote
- app/src/main/res: Layouts e recursos visuais
- app/build.gradle: Configurações do projeto
